Gray:
rgb(100,100,100)

Yellow:
rgb(255,255,0)

NOTES: rgba takes a 4th argument between 0 and 1. An argument of .0 means it will be transparent and 1 means it is solid. Other colors that can be used are hsl() and hsla() which take 3 and 4 arguments. The "h" stands for hue / color, the "s" stands for saturation (grayscale -- colorful) and the "l" stands for lightness / brightness.

Prefered Styles


  • No more than two fonts.
  • Max of 3 colors per page.
  • Use h1 once for main title.
  • Use src:url(../file) for background images in css rather than html.
  • No more than two columns.
  • Link new fonts as html link.
  • Use classes and occasional id's.
